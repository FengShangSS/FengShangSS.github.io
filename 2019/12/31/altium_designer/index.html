Altium Designer19

Posted on 2019-12-31 Views:

破解及汉化

1.将 shfolder.dll复制到AD目录下

2.打开AD，在licenses manager里导入 .alf 文件即可破解

3.点击右上角齿轮标志，在System中找到General，勾选Localization选项，重启

元件库

已有元件库

1.下载元件库https://designcontent.live.altium.com/#UnifiedComponents

2.点击右上角齿轮标志，在Data Management中找到File-based Libraries导入已有元件库

其他设置

四层PCB

1.建立PCB文件

2.选择Designe –> Layer Stack Manger（设计 –> 层叠管理器）

3.右键Top Layer，insert layer below

原理图修改纸张大小

右键，选择原理图优先选项，在General中修改图纸尺寸

元器件自动编号

工具–>标注–>原理图标注
