5.相关问题

Q：执行conda create -n 环境的名字 python=版本号时，出现CondaHTTPError: HTTP 000 CONNECTION FAILED for url https://repo.anaconda.com/pkgs/r/linux-64/repodata.json.bz2

A：更新conda……直接翻墙解决或者更换源

更新了好久才发现虚拟机没有共享到主机的网络，没翻墙，难怪下包下得那么慢……

so研究了一下虚拟机如何共享主机网络

1.开启网络共享

控制面板\所有控制面板项\网络连接里将WLAN的共享打开

2.主机打开cmd获取ip

3.shadowsocks

主机的ss设置成允许其他设备接入

4.虚拟机

网络设置里将主机的IP复制过来，端口为1080，浏览器设置成使用本地代理

Q:由于更新了python版本，导致conda无法使用

A:因为安装jupyter时自动更新了python版本3.6—>3.7，导致conda无法使用。发现并不需要再安装jupyter，系统已有……
